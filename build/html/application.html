class sixgedgesim.application.Application(name)

An application is defined by a directed acyclic graph (DAG) between modules that generate, compute and receive sink messages.

Args:

name (str): The name must be unique within the same topology.

Returns:

app (Application) : An application object

TYPE_SINK = 'SINK'

e.g an actuator

class sixgedgesim.application.Message(name, source_module, destination_module, instructions=0, bytes=0)

A message is consists by the following values:

Args:

name (str): Name of the message (unique for each application)

source_module (str): Name of the module who sends this message

destination_module (dst): Name of the module who receives this message

instructions (int): Number of instructions to be executed ((by default 0), Instead of MIPS, we use IPt since the time is relative to the simulation units.

bytes (int): the size in bytes (by default 0)
